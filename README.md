# Java Programming Practice ☕

## About This Repository

This repository contains Java programs that I practiced during my 2nd year of college.
It includes different Java concepts such as object-oriented programming, sorting algorithms, threads, exception handling, and basic problem-solving programs.

## Motivation

During my 2nd year, our HOD sir said something to the class that became a strong motivation for all of us:

**"From 12th August, I will execute one program every day after 7:30."**

That statement was not just a normal announcement.
It encouraged the entire class to practice programming consistently.
Inspired by those words, we started writing and executing programs regularly.
This repository is a collection of the programs I practiced during that period of learning and improvement.

## Topics Covered

The repository contains programs related to:

* Basic Java Programs
* Method Overloading
* Inheritance
* Interfaces
* Abstract Classes
* Exception Handling
* Multithreading
* Sorting Algorithms (Selection Sort, Insertion Sort)
* Arrays and Strings
* Number Programs (Palindrome, Factorial, etc.)

## Purpose

* Practice Java programming
* Improve problem-solving skills
* Track learning progress
